# Expected Value: How to Make Decisions Under Uncertainty

*From Non-Trivial*

## The core idea

When we don't know exactly what will happen, how should we decide what to do?

One powerful approach is **expected value reasoning**. The expected value of an action is calculated by multiplying each possible outcome's value by the probability it occurs, then summing across all outcomes.

**Expected Value = Σ (Probability of outcome × Value of outcome)**

For example: If there's a 10% chance an action saves 100 lives, its expected value is 10 lives—even though the actual outcome will be either 0 or 100.

## A simple example

Suppose you have two options for a charitable donation:

- **Option A**: Give to a programme with a guaranteed 1-in-1,000 chance of saving a life (EV = 0.001 lives per dollar)
- **Option B**: Give to a riskier intervention with a 1-in-100 chance of saving a life, but less certain to work (EV = 0.01 lives per dollar)

If the probabilities are accurate, Option B has 10x the expected impact—even though it's "riskier" in the sense of having more variance.

Expected value reasoning helps us take seriously the possibility of high-impact, uncertain outcomes rather than defaulting to what feels safe.

## Why expected value matters for effective altruism

Much of the logic behind effective altruism depends on expected value reasoning. When comparing cause areas, we often cannot know with certainty which will do the most good. But we can reason about:

- How bad is the problem? (Scale)
- What fraction of the problem can be solved? (Tractability)
- How many resources are already going toward it? (Neglectedness)

Even under uncertainty, this kind of reasoning can help us identify that some causes are likely orders of magnitude more impactful than others.

## The "hits-based giving" implication

Expected value reasoning leads naturally to what Holden Karnofsky of Open Philanthropy calls **hits-based giving**: the idea that, like a venture capitalist, you should sometimes fund speculative, high-upside interventions even if most fail—because the expected value of the successes is very high.

If you're 90% confident a charity won't achieve its goal but 10% confident it will prevent a catastrophe, the expected value could still be enormous. A portfolio approach—backing many uncertain but potentially transformative bets—may outperform only funding "sure things."

## Common objections and responses

**Objection: Shouldn't we just do what we know works?**
Response: If we only act on certainties, we systematically underinvest in high-risk, high-reward opportunities. Expected value reasoning captures the fact that probable small wins can be outweighed by improbable large ones.

**Objection: Can we really trust these probability estimates?**
Response: No—but that doesn't make the framework useless. We should hold our estimates with humility, update based on evidence, and acknowledge when we're highly uncertain. But refusing to reason probabilistically doesn't escape the problem; it just means making implicit, unexamined bets.

**Objection: Expected value ignores risk aversion.**
Response: For individuals spending their own money, risk aversion makes sense. But for philanthropic portfolios—especially large ones—diversification makes variance less important, and expected value better captures what we should optimize for.

## Limits of expected value reasoning

Expected value reasoning works best when:
- Probabilities can be meaningfully estimated
- Outcomes can be compared on a common scale
- The stakes don't trigger moral side-constraints

It faces challenges when:
- Probabilities are deeply uncertain (sometimes called "Knightian uncertainty")
- We're comparing incommensurable values
- Scale effects make the calculations feel absurd ("galaxy-brained" reasoning)

A sophisticated approach uses expected value as a guide while also maintaining robustness checks—ensuring that the actions it recommends don't violate basic ethical intuitions or depend too sensitively on uncertain assumptions.

## Key takeaways

1. Expected value = probability × value, summed across outcomes
2. Under uncertainty, this framework helps compare interventions more rigorously than intuition alone
3. It leads to "hits-based" thinking: sometimes the right choice is the risky, uncertain bet with high expected value
4. EV reasoning has real limits and should be combined with robustness and moral common sense
5. Much of EA's cause prioritization logic depends on reasoning about expected value under uncertainty
