#### Text
content::
AI safety is a pre-paradigmatic field — there is no consensus on what the alignment problem fundamentally is, let alone how to solve it. Researchers organize their work into agendas: strategic research roadmaps built on different problem framings and theories of change.

This module focuses on one such agenda: Agent Foundations. Agent Foundations treats AI safety as a formal mathematical problem, seeking universal laws that govern any intelligent goal-directed system regardless of architecture. It builds safety from first principles rather than patching current systems.

This module is a direct preparation for the Agent Foundations program that follows this course.
